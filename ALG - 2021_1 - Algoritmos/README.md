# Como Aprender C (Com menos Livros!)

Se você quer aprender a linguagem C de forma prática e sem se perder em centenas de páginas de teoria maçante, este repositório tem apenas uma grande recomendação: **siga o conteúdo do Wagner Gaspar**.

Em vez de gastar tempo com livros pesados, recomendo fortemente que você utilize estes dois recursos:

* **https://wagnergaspar.com/**: Use a plataforma dele como apoio para pegar os exercícios, materiais complementares e testar seus próprios códigos.
* **https://www.youtube.com/@programeseufuturo/courses**: Área no YouTube com as playlists do Wagner Gaspar. É melhor navegar sequencialmente pelo site dele, lá tem links que te mandam de volta aqui.

Use os livros só como apoio, abra a IDE e vá assistir às aulas. Bons estudos! 💻☕
